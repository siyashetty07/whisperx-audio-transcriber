# WhisperX Audio Transcriber

## Features
- Audio transcription using WhisperX
- Speaker diarization using PyAnnote
- Multi-speaker support
- JSON transcript export
- TXT transcript export
- Audio cleanup pipeline

## Tech Stack
- Python
- WhisperX
- PyAnnote Audio
- Hugging Face
- PyTorch

## Installation

pip install -r requirements.txt

## Usage

python main.py

## Output
- transcript.txt
- diarized.json
- clean.json
